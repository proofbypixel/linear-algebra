1. Vector Space & Span

A vector space $V$ over a field $\mathbb{F}$ is a set where vector addition and scalar multiplication are defined and satisfy specific axioms. I‚Äôm starting to think of Span as the boundary of possibility‚Äîanything outside it is unreachable without new vectors.

$$ \text{span}(\vec{v}_1, \dots, \vec{v}_n) = \{ \sum_{i=1}^{n} \alpha_i \vec{v}_i \mid \alpha_i \in \mathbb{F} \} $$

2. Linear Independence

A set of vectors is linearly independent if the only way to sum them to zero is by setting all coefficients to zero. If one vector can be built from the others, it isn‚Äôt adding a new direction. Independent vectors expand the span; dependent ones don‚Äôt.

Rank is the measure of "true" dimensionality. If a dataset has 10 features but the feature matrix has a Rank of 3, the model effectively operates in 3 dimensions. 2

Insight: Rank-Nullity Information is either preserved (Rank) or lost (Kernel), never both.

1. Linear vs. Affine

A pure Linear model forces predictions to pass through the origin $(0,0)$. An Affine model allows a shift via an intercept (bias term).

Log 003: Norms & Inner Products

18 Dec 2025

Norms measure size, but Inner Products allow vectors to "talk" to each other (measure alignment). The most familiar is the Euclidean norm:

$$ \|\vec{v}\|_2 = \sqrt{\vec{v}^\top \vec{v}} $$

The Geometry of Interaction

An inner product introduces geometry (angles) into a vector space. It allows us to define Orthogonality. If $\langle \vec{u}, \vec{v} \rangle = 0$, the vectors are orthogonal. Zero interaction means zero alignment.

Log 004: Orthogonality & Error

19 Dec 2025

In Least Squares regression, we try to approximate a target vector $\vec{y}$ using our feature space. The "Error" vector is simply the part of $\vec{y}$ that lives in the Orthogonal Complement. 3

3. Why Orthogonal? If the error weren't orthogonal, it would mean there is still some "direction" left in the features that explains the target, and we could reduce the error further.
